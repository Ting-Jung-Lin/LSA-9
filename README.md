# 以財換柴，錢拿出來

## 動機發想
* 我們回家有時候都會把零錢隨意地放在桌上，十分凌亂，所以我們想請療癒的柴柴幫我們收集零錢，讓零錢集中在一個地方，當然也有抖內箱的功能~看到可愛的柴柴誰不會想投個錢看看他呢?

* 既然是抖內箱，就想看看是哪位金主貢獻了無用的(寶貴的)錢錢~可愛的柴柴會依依記錄下來，但不會感謝你。柴柴只會拿錢，不會做事。
